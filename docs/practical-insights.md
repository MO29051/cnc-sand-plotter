# Practical Insights

## 🧠 Observations from Implementation

### Motion Quality

* Lower speeds produced smoother and clearer patterns
* Higher speeds caused sand disturbance and reduced accuracy

---

### Mechanical Behavior

* System stability directly affects drawing quality
* Misalignment leads to distorted patterns

---

### G-code Execution

* Continuous paths produce better results than segmented movement
* Sudden direction changes affect smoothness

---

## ⚠️ Challenges Faced

* Tuning feed rate for optimal performance
* Achieving consistent surface conditions
* Ensuring smooth motor operation

---

## 🎯 Key Takeaways

* CNC systems rely heavily on calibration and tuning
* Mechanical design is as important as control logic
* External tools simplify complex pattern generation

---

## 🔮 Possible Improvements

* Improve frame rigidity
* Optimize motion speed
* Implement better path smoothing techniques
