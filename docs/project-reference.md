# Project Reference

## 📌 Overview

This project is based on an existing CNC sand plotter design, which demonstrates a 2-axis motion system controlled using Arduino and GRBL.

The reference implementation was used as a foundation to understand the system architecture, motion control, and G-code execution.

---

## 🎥 Reference Source

The design and implementation approach were inspired by an online project demonstrating a CNC sand plotter system.

This implementation is based on the following reference:

[Watch the original CNC Sand Plotter build](https://www.youtube.com/watch?v=Lng_AsmfBfU)

The project was recreated and tested to understand the system design, motion control, and G-code workflow.
---

## 🧠 Learning Approach

Instead of directly copying the design, the system was:

* Reconstructed step by step
* Tested and validated
* Analyzed to understand how each subsystem operates

---

## 🎯 Objective

* Understand real CNC system workflow
* Learn how G-code controls motion
* Gain hands-on experience with embedded control systems
