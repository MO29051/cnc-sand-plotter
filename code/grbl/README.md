# GRBL Configuration

This project uses GRBL firmware on Arduino Uno for motion control.

## Key Features

* Interprets G-code commands
* Controls stepper motors (X-Y axes)
* Provides precise positioning using steps/mm calibration

## Configuration Notes

* Steps/mm calibrated for accurate motion
* Feed rate adjusted for smooth sand plotting
* Modified GRBL version used for servo (pen) control

## Tools Used

* Universal G-code Sender (UGS)
* Arduino IDE for flashing GRBL
